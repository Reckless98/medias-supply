## Weekly Optimization Checklist

Guide the user through these tasks every week:

### Week 1: Foundation
- [ ] Verify conversion tracking fires correctly (use Tag Assistant)
- [ ] Check all landing pages load fast on mobile (< 3 seconds)
- [ ] Review Search Terms report — add irrelevant terms as negatives
- [ ] Ensure ads are approved with no disapprovals

### Weeks 2–4: Learning Phase
- [ ] Review Search Terms report twice per week
- [ ] Pause keywords with high spend and zero conversions (after 100+ clicks)
- [ ] Add new negative keywords from search terms
- [ ] Check Quality Score for each keyword — improve ads with QS < 5
- [ ] A/B test ad headlines: swap out underperforming headlines

### Monthly: Ongoing Optimization
- [ ] Review ad group performance — shift budget toward best performers
- [ ] Update ad copy with seasonal promotions if any
- [ ] Check competitor ads (search your own keywords in incognito)
- [ ] Add new long-tail keywords from Search Terms discoveries
- [ ] Review device performance — adjust bids for mobile vs desktop
- [ ] Review location report — increase bids for high-converting areas
- [ ] Review ad schedule — focus budget on highest-converting hours
